# Data

This folder contains the Kaggle datasets used for this research, all related to Indonesian snake species identification and detection.

## Datasets

### 1. Snake Image Dataset Indonesia
- **Link:** https://www.kaggle.com/datasets/muhffikkri/snake-image-dataset-indonesia
- **Description:** A raw image dataset of Indonesian snake species, organized into one folder per species (e.g. `Acanthophis_antarcticus`, `Ahaetulla_prasina`, `Boiga_dendrophila`, etc.). It covers around 293 species folders and contains roughly 12.6k `.jpg` images, with a total size of about 5.79 GB (Version 1). This dataset serves as the primary raw image source for the pipeline.

### 2. Snake Coordinate Dataset
- **Link:** https://www.kaggle.com/datasets/muhffikkri/snake-coordinate-dataset
- **Description:** A tabular (metadata) dataset records linking each snake image to its taxonomic and location information. Columns include:
  - `full_name` — full scientific name (245 unique values)
  - `genus` — genus name (85 unique values)
  - `species` — species name (224 unique values)
  - `coords` — geographic coordinates (latitude, longitude) of the recorded sighting
  - `locality` — location/region name (e.g. "Jawa Barat, ID", "Indonesia")
  - `bisa` — venom status label, with two classes: *Tidak berbisa* (non-venomous, ~81%) and *Bisa* (venomous, ~19%)
  - `image_folder_name` — corresponding species folder name in the Snake Image Dataset Indonesia
  - `image_path_full` — full image path reference (e.g. `/kaggle/input/snake-image-dataset-indonesia/Ahaetulla_Prasina/...`)
  
  This dataset is used to map each image to its species, genus, venom status, and collection locality.

### 3. Snake YOLO Cropped
- **Link:** https://www.kaggle.com/datasets/muhffikkri/snake-yolo-cropped
- **Description:** A processed dataset containing cropped snake images generated by running YOLO object detection on the raw images, so each crop is focused on the snake's body region only. The dataset contains:
  - `crops/` — ~12.3k `.jpg` cropped image files
  - `convnext_large_finetuned.pth` — a fine-tuned ConvNeXt-Large model checkpoint
  - `huggingface_repos.json` — reference file listing related Hugging Face model repositories
  
  This dataset is used for the downstream classification stage after object detection/cropping.
